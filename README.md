# Filesystem project

## Build the project

1. Install make for Windows
2. Install g++ for Windows
3. run ```make``` in the root folder of the project
### The project can also be built with just using:
  ```g++ src/* -std=c++17 -o main.exe```

## Run the cli
run ```.\main.exe```

## Run the unit tests
1. Build the project (steps above)
2. run ```.\main.exe --test```
